## 当鼠标移动到圆形封面或者三个播放键

圆形封面会旋转同时会有歌曲的相关信息显示

同时进度条可以拖动

## 单击圆形转盘后会显示歌曲列表

在歌曲列表里可以自由选择歌曲和模式
